## Style Guidelines:

- Primary color: Calm blue (#64B5F6), promoting trust and reliability.
- Background color: Light gray (#F0F4F8), provides a neutral canvas.
- Accent color: Soft orange (#FFAB91) for highlighting key actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look, for both headlines and body text.
- Simple, line-based icons for common actions.
- Clean, grid-based layout with a focus on readability and ease of use.
- Subtle transitions for actions like adding blocks and publishing the page.
